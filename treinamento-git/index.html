Treinamento GIT

1- Quais os comandos para configurar usuário e email no git?

Resposta:git config --global user.name "Nome Sobrenome" git config --global user.email "email@dominio.com"

2- Qual a diferença entre git e github?

Resposta: GIT é o software que se instala na maquina e faz integracao com um repositorio como github. Já o GitHub é a plataforma web para armazenamento dos codigos, controle de versao e colaboracao

3- O que faz o comando commit?

Resposta: Commit é um save point ou um ponto salvo de retorno caso seja necessario.

4- O que faz o comando push?

Resposta: Comando Push serve para enviar as alteracoes em commit do repositorio local para o remoto, como a plataforma GitHub.

5- O que faz o comando pull?

Resposta: O comando Pull serve para puxar (Baixar e atualizar) o conteudo do repositorio remoto para o local.
